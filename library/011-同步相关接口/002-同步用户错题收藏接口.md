# 同步用户错题收藏接口

> 维护人员：王彩霖

> 创建时间：2017年11月27日

## 接口详情

### 请求地址
```
/api/questionsuser/saveQuestionsUser
```

### 请求类型
GET POST

| 参数名 | 类型 | 必填 | 描述 | 默认值 | 参考值 |
| --- | :---: | :---: | --- | --- | --- |
| uid | number | 是 | 用户ID | - | 1 |
| qid | number | 是 | 试题ID | - | 1 |
| type | number | 是 | 0消防员 1消防工程师 | - | 1 |
| errors | number | 否 | 1全部错题 2知错能改 3两者都有 | - | 3 |
| collect | number | 否 | 收藏 | - | 1 |

### 请求参数
无

### 正确响应
```javascript
{
    "success": true,
    "code": 0,
    "msg": "同步成功",
    "data": null
}
```

### 错误响应
```javascript
{
    "success": true,
    "code": 0,
    "msg": "同步失败",
    "data": null
}
```
